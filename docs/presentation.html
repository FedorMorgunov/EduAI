AI-powered платформа для адаптивного обучения
Уроки · Квизы · REST API

Фёдор Моргунов  ·  2026  ·  v0.1.0

Проблема и решение

❌ Проблема

  • Статичные учебные материалы не адаптируются под уровень студента
  • Нет быстрого способа генерировать квизы по теме
  • Контент разрознен — нет единого API для интеграции

✅ Решение

  • Единый REST API для уроков и квизов с фильтрацией по теме и сложности
  • Готов к подключению AI-генерации контента (Claude API)
  • Чистая архитектура — сервисный слой легко расширяется
